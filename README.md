# ⏳ Time Travel Portal

A fun web-based Time Travel simulation built using HTML, CSS, and JavaScript.

Users can enter a year between **1900 and 2050** and explore information from the past or the future.

---

## 🚀 Features

- Time Travel simulation
- Past and Future pages
- Loading animation
- Sound effects
- Local JSON database
- Responsive interface
- Pure HTML, CSS and JavaScript

## ▶ How to Run

1. Download or clone the repository.
2. Open `index.html` in your browser.
3. Enter a year between **1900–2050**.
4. Click **Travel**.
